# Swiss Knife

**Swiss Knife** es una aplicación multipropósito desarrollada en Java, diseñada para reunir diversas herramientas útiles en un solo lugar. Su interfaz está inspirada en un smartwatch, con un carrusel de herramientas que permite una navegación fácil y rápida.

## Funcionalidades

### Calculadora
Swiss Knife incluye una calculadora básica que permite realizar operaciones rápidas y sencillas como:
- Sumar
- Restar
- Dividir
- Multiplicar

### Moneda
Esta herramienta permite realizar lanzamientos de moneda aleatorios, útil para decisiones rápidas. El contador muestra cuántas veces ha salido "cara" o "cruz" y cuantas veces se a lanzado la moneda en total, durante las sesiones de uso.

### Visualizador de HTML
Permite escribir y visualizar código mixto de HTML y Markdown. Los usuarios pueden trabajar cómodamente con etiquetas HTML, pero el soporte de Markdown es limitado a HTML básico (no soporta HTML5). 

- *Reiniciar a HTML5*: Comienza con una estructura básica de HTML5.
- *Reiniciar a HTML*: Inicia con una estructura básica de HTML.
- *Reiniciar a Markdown*: Inicia con una estructura básica de Markdown.

Además, incluye opciones para exportar el contenido:
- *Exportar a HTML*: Exporta el código en formato HTML.
- *Exportar a PDF*: Exporta el código en formato PDF.

### Juego de Encontrar
En este juego, la aplicación selecciona un número entre 1 y 100, y el usuario debe adivinarlo. La aplicación ofrece pistas como:
- *El número es mayor que* N
- *El número es menor que* N

El objetivo es adivinar el número en el menor número de intentos posibles.

### Repetidor de frases
Una funcionalidad sencilla, se escribe una frase, se decide cuantas veces repetirla (1-800) y se repetira esa cantidad de veces y estara lista para copiar y lo pegues donde quieras

Funcionalidad extra:
- *@N* genera numeros en orden por cada repetición

### Generador de Paletas
Una funcionalidad para generar paletas de colores, usando como base los "RGB" y los "HEX", cuando los ingreses se mostrara automaticamente en el previsualizador de color junto con su codigo HEX, a medida que escribas los codigos se acuatualizara automaticamente el previsualizador de colores

tienes 8 botones disponibles para que puedas generar una paleta basado en el color elegido:
- *Complementario* dos colores
- *Ternario* Tres colores
- *Cuadernario* Cuatro colores
- *Pentadrico* Cinco Colores
- *Sombra* Nueve tonos empezando del color base y llegando al mas obscuro
- *Tinte* Nueve tonos empezando del color base y llegando al mas claro
- *Monocromatico* Nueve todos empezando por el color mas claro llegando al color base y llegando al mas obscuro
- *Analgo* Los colores mas cercanos a los lados del color base

### Mathmixer
Varias funciones Matematicas unidas en una sola función, las entradas varian entre una entrada de datos y dos entradas de datos dependiendo de la función matematica elegida

Funciones disponibles:
- *Tablas* para sacar las tablas de multiplicar de un numero
- *Multiplo* para sacar los numeros por los que es divisible el numero ingresado
- *Factorial* calcula el factorial del numero ingresado, tiene el limite de numero de "*9999*"
- *3N+1* la conjetura de *collatz*, si el numero es par se divide entre 2 y si es impar se multiplica por tres y se suma uno, esto se hara con los resultados, en cierto momento los calculos daran *1*
